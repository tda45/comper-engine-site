## 🔧 Kullanım Rehberi

### Asset Erişimi
```cpp
// C++ ile asset erişimi
#include "comper_sdk.h"

// 4K background al
void* background = AssetLibrary::Get4KBackground("abstract_red_001.jpg");

// SVG ikon al
std::string icon = AssetLibrary::GetSVGIcon("home.svg");

// Font yükle
void* font = AssetLibrary::GetFont("ComerSans-Regular.ttf");

// Animasyon oynat
AssetLibrary::PlayAnimation("neon_glow_pulse.json", target_element);
```

### Asset Optimizasyonu
- Tüm 4K görseller WebP formatında sıkıştırılmış
- SVG ikonlar optimize edilmiş ve minified
- Fontlar subset'lenmiş ve WOFF2 formatında
- Animasyonlar Lottie formatında (JSON)

### Memory Management
- Asset'ler lazy loading ile yüklenir
- Bellek kullanımı otomatik optimize edilir
- Cache sistemi ile hızlı erişim
- Garbage collection ile memory leak önleme

---

**Not:** Asset kütüphanesi SDK Manager tarafından otomatik yönetilir. Asset'leri doğrudan değiştirmek yerine SDK Manager API'ını kullanın.
