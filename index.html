SDK Core API

Comper Engine'in temel motoru ve çekirdek işlevsellikleri.

#include "Includes/Core/comper_core.h"
#include "Includes/SDK/comper_sdk.h"

using namespace ComperCore;
using namespace ComperSDK;

class MyBrowser : public ComperEngine {
public:
    void Run() {
        // Night Red Ultra temasını uygula
        NightRedUltra::EnableNeonGlow(true);
        NightRedUltra::SetNeonIntensity(0.8f);
        NightRedUltra::SetGlowColor(230, 0, 0);
        
        // Sayfayı yükle
        NetworkResponse response = LoadPage("https://example.com");
        
        if (response.IsSuccess()) {
            auto document = ParseHTML(response.body);
            RenderPage(document);
        }
    }
};

Özellikler

  • Engine Management: Başlatma, kapatma, konfigürasyon
  • Theme System: Night Red Ultra entegrasyonu
  • Resource Management: Bellek optimizasyonu ve kaynak yönetimi
  • Error Handling: Detaylı hata yönetimi ve logging

SDK Network API

HTTP/HTTPS istekleri, WebSocket bağlantıları ve ağ yönetimi.

NetworkManager network;
network.Initialize();

NetworkResponse response = network.Get("https://example.com");
if (response.IsSuccess()) {
    std::cout << "Content: " << response.body << std::endl;
}

// Async istek
network.AsyncGet("https://api.example.com/data", 
    [](const NetworkResponse& response) {
        // Callback fonksiyonu
        std::cout << "Async response received!" << std::endl;
    });

Özellikler

  • HTTP/HTTPS Support: Modern protokol desteği
  • Connection Pooling: Performans için bağlantı havuzu
  • Async Operations: Asenkron istek desteği
  • SSL/TLS Security: Güvenli bağlantılar
  • Cookie Management: Otomatik cookie yönetimi

SDK Render API

2D/3D rendering, Night Red Ultra grafik motoru ve GPU acceleration.

Renderer renderer;
renderer.Initialize(window_handle);

// Night Red Ultra efektleri
NightRedUltra::EnableNeonGlow(true);
NightRedUltra::SetNeonIntensity(0.9f);

// Çizim işlemleri
renderer.Clear(Color(0.02f, 0.02f, 0.02f, 1.0f));
renderer.DrawFilledRectangle(Rectangle(100, 100, 200, 50), 
                              Color(0.9f, 0.0f, 0.0f, 1.0f));
renderer.DrawText("Hello Comper!", Point(150, 150), 
                  Color(1.0f, 1.0f, 1.0f, 1.0f));
renderer.Present();

SDK Browser API

Browser browser;
browser.Initialize();

// Sekme yönetimi
std::string tab1 = browser.CreateTab("https://google.com");

SDK Download API

Multi-threaded indirme yönetimi, pause/resume desteği ve güvenlik kontrolleri.

DownloadManager download_manager;
download_manager.Initialize();

// İndirme ekleme
std::string download_id = download_manager.AddDownload(
    "https://example.com/file.zip", 
    "download.zip", 
    DownloadPriority::HIGH
);

// Progress takibi
download_manager.on_download_progress = 
    [](const DownloadItem& download) {
        std::cout << "Progress: " 
                  << download.GetFormattedProgress() << std::endl;
    };

// İndirme kontrolü
auto download = download_manager.GetDownload(download_id);
if (download->IsDownloading()) {
    download->Pause();
    // Daha sonra devam ettir
    download->Resume();
}

Özellikler

  • Multi-threaded: Eşzamanlı indirme desteği
  • Pause/Resume: İndirme kontrolü
  • Progress Tracking: Real-time progress ve ETA
  • File Verification: Bütünlük kontrolü
  • Malware Scanning: Güvenlik taraması
