Core API Genel Bakış

Comper Engine Core API, tarayıcı geliştirmek için temel işlevsellikleri sağlar. Network, rendering, HTML parsing ve daha fazlasını içerir.

Temel Kullanım

#include "Includes/Core/comper_core.h"

using namespace ComperCore;

int main() {
    ComperEngine engine;
    
    // Motoru başlat
    if (engine.Initialize()) {
        // Sayfa yükle
        NetworkResponse response = engine.LoadPage("https://example.com");
        
        if (response.IsSuccess()) {
            // HTML'i parse et
            auto document = engine.ParseHTML(response.body);
            
            // Sayfayı render et
            engine.RenderPage(document);
            
            // DevTools'u göster
            engine.ShowInspector();
        }
        
        // Event loop
        engine.Run();
    }
    
    engine.Shutdown();
    return 0;
}

ComperEngine Sınıfı

Comper Engine'in ana motor sınıfı. Tüm temel işlevsellikleri yönetir.

bool Initialize()
Comper Engine motorunu başlatır. Network, renderer ve diğer bileşenleri kurar.
Dönüş Değeri:
Başarılı olursa true, aksi halde false
void Shutdown()
Comper Engine motorunu güvenli bir şekilde kapatır. Tüm kaynakları serbest bırakır.
NetworkResponse LoadPage(const std::string& url)
Belirtilen URL'den web sayfasını yükler.
url:
const std::string&
Yüklenecek web sayfasının URL'si
Dönüş Değeri:
NetworkResponse nesnesi
std::shared_ptr<DOMDocument> ParseHTML(const std::string& html)
HTML string'ini DOM dokümanına parse eder.
html:
const std::string&
Parse edilecek HTML string'i
Dönüş Değeri:
DOM dokümanının shared pointer'ı
void RenderPage(std::shared_ptr<DOMDocument> document)
DOM dokümanını ekrana render eder.
document:
std::shared_ptr<DOMDocument>
Render edilecek DOM dokümanı
void Run()
Comper Engine event loop'unu başlatır. Programın ana döngüsüdür.
void ShowInspector()
Comper Inspector (DevTools) penceresini gösterir.
void HideInspector()
Comper Inspector (DevTools) penceresini gizler.
bool IsInspectorVisible() const
Inspector penceresinin görünürlüğünü kontrol eder.
Dönüş Değeri:
Görünürse true, değilse false

Örnekler

Core API kullanımı için pratik örnekler.

Basik Tarayıcı Örneği

#include "Includes/Core/comper_core.h"

using namespace ComperCore;

class SimpleBrowser : public ComperEngine {
public:
    void LoadWebsite(const std::string& url) {
        std::cout << "Loading: " << url << std::endl;
        
        // Sayfayı yükle
        NetworkResponse response = LoadPage(url);
        
        if (response.IsSuccess()) {
            std::cout << "Page loaded successfully!" << std::endl;
            std::cout << "Content length: " << response.content_length << std::endl;
            
            // HTML'i parse et
            auto document = ParseHTML(response.body);
            if (document) {
                std::cout << "HTML parsed successfully!" << std::endl;
                
                // Sayfayı render et
                RenderPage(document);
                
                // DevTools'u göster
                ShowInspector();
            }
        } else {
            std::cout << "Failed to load page: " << response.error_message << std::endl;
        }
    }
};

int main() {
    SimpleBrowser browser;
    
    // Motoru başlat
    if (browser.Initialize()) {
        // Web sitesi yükle
        browser.LoadWebsite("https://example.com");
        
        // Event loop
        browser.Run();
    }
    
    return 0;
}

Network İsteği Örneği

#include "Includes/Core/comper_core.h"

using namespace ComperCore;

int main() {
    NetworkManager network;
    
    // Network manager'ı başlat
    if (network.Initialize()) {
        // Senkron istek
        NetworkResponse response = network.Get("https://api.example.com/data");
        
        if (response.IsSuccess()) {
            std::cout << "Response: " << response.body << std::endl;
            std::cout << "Status: " << response.status_code << std::endl;
        }
        
        // Asenkron istek
        network.AsyncGet("https://api.example.com/async", 
            [](const NetworkResponse& response) {
                if (response.IsSuccess()) {
                    std::cout << "Async response: " << response.body << std::endl;
                } else {
                    std::cout << "Async error: " << response.error_message << std::endl;
                }
            });
        
        // Event loop (basit)
        std::this_thread::sleep_for(std::chrono::seconds(5));
    }
    
    return 0;
}

Rendering Örneği

#include "Includes/Core/comper_core.h"

using namespace ComperCore;

int main() {
    ComperEngine engine;
    Renderer renderer;
    
    // Motor ve renderer'ı başlat
    if (engine.Initialize() && renderer.Initialize(nullptr)) {
        // Ekranı temizle
        renderer.Clear(Color(0.02f, 0.02f, 0.02f, 1.0f));
        
        // Dikdörtgen çiz
        Rectangle rect(100, 100, 200, 50);
        renderer.DrawFilledRectangle(rect, Color(0.9f, 0.0f, 0.0f, 1.0f));
        
        // Metin çiz
        renderer.DrawText("Hello Comper Engine!", Point(150, 150), 
                          Color(1.0f, 1.0f, 1.0f, 1.0f));
        
        // Ekrana gönder
        renderer.Present();
        
        // Event loop
        engine.Run();
    }
    
    return 0;
}
